Coding Projects

I tend to write a lot of code when I have an idea for something and sometimes I come back to it when I want to write a lot. My current projects that I am ACTIVELY working on right now are listed below. My goal is to have them all completed.

  • Personal Website - This website is pretty simple and I often am writing JavaScript functions and messing around with it. I often am using it as a sandbox environment to test the latest tech on as opposed to as a static resource so if things look a little funky, that explains why.
  • StudyLock - This is my current biggest project that I am working on with a friend and fellow software developer, Jacob Chapman for both of our Senior Design project. It is currently under development and is being built using Meteor.js (which I love), React Native, MongoDB, Modulus, and many more.
  • Jarvis - This one is probably my proudest achievement so far when it comes to coding. I have built this system that reads out my system information using Microsft voice. I built this to overcome issues when having an overclocked system. Instead of just having something that displays my CPU temperature this will actually speak to me. Built using C#, Microsft Speech API, System Readings.
